# DocuSign

## Requirements
Configuration requires [Node.js](https://nodejs.org/) v8+ to run.

## Installation
Go to /markup directory
```bash
cd DSHR/frontend/markup
```

Use the package manager [npm](https://www.npmjs.com/) to install dependencies.
```bash
npm install
```

## Build & watch

You can use
```bash
npm run build
```

or

```bash
gulp
```
